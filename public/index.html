API 엔드포인트

서버 상태 확인 API

GET /api/games

사용 가능한 게임 항목 목록 조회

POST /api/start

새 게임 시작

{
  "item_id": 1  // (선택 사항) 특정 게임 항목 ID
}
POST /api/ask

AI에게 질문하기

{
  "game_id": "game_xxx",
  "message": "사용자 질문"
}
POST /api/end

게임 종료

{
  "game_id": "game_xxx"
}
